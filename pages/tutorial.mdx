# How to use this site

This website is aimed to document real science over opinion in the health space. With the power of AI, this website pulls information from reputible sources and documents those sources under each article.
We have created this site in a documentation style as we feel the information can be read in any order.

## Where to start

Start where you feel comfortable. Search for the content that matters most to you. You can use the search bar in the top right of the screen to search for any piece of information, or simply find the information in the navigation on the left.

The Information has been split into 4 sections; Body, Mind, Wealth and Family. These 4 areas in a man's life are the most important in this modern world.
